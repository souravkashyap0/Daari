> Just route it.

Daari is a lightweight, plug-and-play JavaScript router 
for small applications. No dependencies, no npm, no build 
tools — just drop it in and go.

## Why Daari?
- Zero dependencies
- Pure vanilla JavaScript
- Simple API
- Plug and play — just include one file
